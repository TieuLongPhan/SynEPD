Welcome to SynEPD's documentation!
==================================

SynEPD (Synthetic Electron Pushing Database) is an advanced framework and database for analyzing organic reaction mechanisms, identifying reaction centers, mapping atom numbers, and matching subgraph isomorphism for elementary reactions.

Features:
---------
* **Reaction center detection**: Auto-detect reaction centers and draw canonical 2D/3D representations.
* **Atom mapping validation**: Ensure complete hydrogen balance and correct atom mapping.
* **Web UI & Knowledge Graph**: Browse reaction hierarchies and inspect chemical properties interactively.
