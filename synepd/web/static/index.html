Quick Start Guide

  • Select a reaction category from the Taxonomy Tree tab.
  • Search via Case ID (e.g. polar06_699) or class names in the search bar.
  • Enter custom SMILES reactions (e.g. CC[O-].[NH4+]>>CCO) to trace automated arrow projections.
